Market simulation
Market Simulation Model Simulates daily stock price movements using Geometric Brownian Motion (GBM) — the industry-standard model for asset prices.

Each trading day, the portfolio changes by:
exp((μ − σ²/2)·dt + σ·√dt·Z)

where μ is your return rate, σ = 16% (approx. historical S&P 500 volatility), dt = 1/252 trading year, and Z is a random draw from a standard normal distribution.

The expected long-run outcome matches steady growth, but individual paths swing — as in real markets. Each re-run produces a different path.
